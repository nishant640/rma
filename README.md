# ⚽ Real Madrid Fan Website

A modern React-based fan website dedicated to Real Madrid, featuring news, players, matches, and more.

🔗 **Live Site:**  
https://nishant640.github.io/rma/

---

## 🚀 Features

- Homepage with featured content  
- News section with latest updates  
- Players page with squad breakdown  
- Matches page (results + upcoming fixtures)  
- Login page UI  
- Fully responsive design  
- Clean and modern interface  

---

## 🛠️ Tech Stack

- React  
- React Router  
- CSS  
- GitHub Pages  

---

## 📦 Installation & Setup

```bash
git clone https://github.com/nishant640/rma.git
cd rma
npm install
npm start
```

---

## 🚀 Deployment

```bash
npm run deploy
```
